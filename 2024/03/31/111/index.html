如何搭建自己的代理池

搭建过程：

Redis 下载：https://github.com/tporadowski/redis/releases

下载成功后：输入命令：

``redis-server.exe redis.windows.conf`， 启动Redis服务

另外再打开一个cmd窗口，输入命令：redis-server --service-install redis.windows.conf，安装Redis到Windows服务

然后 提示安装服务 同意后启动

proxy_pool下载：Releases · jhao104/proxy_pool (github.com)

下载对应的包：pip install -r requirements.txt

有些包版本可能有问题 需要重新下载

执行：python proxyPool.py schedule && python proxyPool.py server

也可以写一个脚本 start_proxy.sh

访问 127.0.0.1：5010/get 即可看到获取到的代理ip
